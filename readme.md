## Desenvolvimento da Inteligência Artificial com Python 

Este repositório foi criado para o desenvolvimento da inteligência artificial utilizando python para o desenvolvimento do API do 6° semestre.

## Pré-requisitos 🛠️
- **Python 3.10 ou superior**
- **Visual Studio Code**

## Para rodar a aplicação 🖥️

### Windows 🪟
- Instalação das bibliotecas utilizadas:
  - `pip install -r requirements.txt`
- Rodar a aplicação:
  - `python ia-camera.py`
 

### Linux 🐧
Para simplificar o processo de configuração do ambiente no Linux, fornecemos um script chamado `ambiente.sh`. Este script automatiza a instalação de todas as dependências necessárias para executar a aplicação, incluindo o Python e as bibliotecas requeridas.
- Execução do script:
  - `./ambiente.sh`
- Rodar a aplicação:
  - `python ia-camera.py`
